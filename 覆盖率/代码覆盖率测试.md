# 代码覆盖率

工具：gcov lcov

## 终局生成

执行命令：
```shell
./sudoku -c 10
```

## 游戏生成

执行命令：

```shell
./sudoku -n 1000
```

![游戏生成1](游戏生成1.jpg)

```
./sudoku -n 1000 -m 3
```

![游戏生成2](游戏生成2.jpg)

```shell
./sudoku -n 1000 -r 30~50
```

![游戏生成3](游戏生成3.jpg)

```shell
./sudoku -n 1000 -u
```

![游戏生成4](游戏生成4.jpg)

```shell
./sudoku -n 1000 -m 3 -r 20~55 -u
```

![游戏生成5](游戏生成5.jpg)
